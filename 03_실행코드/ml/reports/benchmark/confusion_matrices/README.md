# Confusion Matrices

이미지 파일: confusion_matrices.png

공통 벤치마크 기준
- 데이터: 지역사회건강조사 재범주화 데이터, 19세 이상 39세 이하 청년층 표본 134,672건.
- 참고 라벨: PHQ-9 합산점수 10점 이상을 Reference high, 10점 미만을 Reference low로 정의하였다.
- 클래스 분포: Reference high는 5,468건으로 전체의 4.1%이고, Reference low는 129,204건으로 전체의 95.9%이다.
- 벤치마크 목적: Logistic Regression, Random Forest, XGBoost, LightGBM을 같은 데이터 분할과 같은 전처리 조건에서 비교해 어떤 모델이 선별 목적에 적합한지 확인하는 것이다.
- 해석 주의: 벤치마크 결과는 모델 비교용이다. 실제 웹사이트에 연결된 운영 모델은 ml/artifacts/risk_model.joblib의 LightGBM이며 운영 임계값은 0.590이다. 벤치마크 LightGBM은 임계값 0.588 기준이므로 운영 모델과 매우 가깝지만 완전히 같은 산출물은 아니다.

그림이 보여주는 것
혼동행렬은 모델이 맞힌 사례와 틀린 사례를 네 칸으로 나눈 표이다. 세로축은 실제 라벨, 가로축은 모델 예측이다. 각 모델이 실제 Low와 실제 High를 어떻게 분류했는지 건수로 보여준다.

벤치마크 용어 설명
- TN: 실제 Low를 Low로 맞힌 경우이다.
- FP: 실제 Low인데 High로 잘못 예측한 경우이다. 상담 경고가 과하게 나가는 경우로 볼 수 있다.
- FN: 실제 High인데 Low로 놓친 경우이다. 선별 모델에서 특히 줄이고 싶은 오류이다.
- TP: 실제 High를 High로 맞힌 경우이다.

결과 해석
LightGBM은 TN 21,656건, FP 4,185건, FN 307건, TP 787건을 보였다. 즉 실제 High 1,094건 중 787건을 찾아냈고 307건은 놓쳤다. Logistic Regression은 TP 795건으로 가장 많은 High를 잡았고, XGBoost는 FP가 3,952건으로 비교 모델 중 상대적으로 적었다.

보고서에 넣을 문장
"혼동행렬 기준으로 LightGBM은 실제 고위험군 1,094건 중 787건을 고위험군으로 예측했으며, 307건은 놓쳤다. 이는 고위험군 선별을 위해 recall을 확보하는 대신 일부 저위험군이 고위험군으로 분류되는 trade-off가 있음을 보여준다."

주의점
FP가 많다고 해서 모델이 무조건 나쁜 것은 아니다. 우울 고위험군 선별에서는 놓치는 사례를 줄이는 것이 중요할 수 있지만, 사용자에게 불필요한 불안을 주지 않도록 결과 표현은 조심해야 한다.
