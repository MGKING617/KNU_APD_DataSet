# 보고서용 벤치마크 표

이미지 파일: publication_benchmark_tables.png

공통 벤치마크 기준
- 데이터: 전문가 상담 후 재전처리한 지역사회건강조사 데이터, 19세 이상 39세 이하 청년층 표본 134,672건.
- 참고 라벨: PHQ-9 합산점수 10점 이상을 Reference high, 10점 미만을 Reference low로 정의하였다.
- 클래스 분포: Reference high는 5,468건으로 전체의 4.1%이고, Reference low는 129,204건으로 전체의 95.9%이다.
- 변수 변경: 가구원 수는 인원수 그대로 쓰지 않고 1인 가구 여부(single_person_household)로 재범주화하였다.
- 제외 변수: 주관적 건강인식, 체형인식(마름/비만), 아침식사 빈도, 교육수준은 전문가 상담 내용을 반영해 입력 변수에서 제외하였다.
- 벤치마크 목적: Logistic Regression, Random Forest, XGBoost, LightGBM을 같은 데이터 분할과 같은 전처리 조건에서 비교해 어떤 모델이 선별 목적에 적합한지 확인하는 것이다.
- 임계값 기준: 이번 전문가 상담 후 실험에서는 모든 모델의 예측 확률 임계값을 0.590으로 고정하였다.
- 해석 주의: 이 폴더의 결과는 기존 ml 폴더를 대체한 것이 아니라, 전문가 상담 이후 변수 구성을 바꿔 다시 돌린 별도 비교 실험이다.

그림이 보여주는 것
이 이미지는 보고서나 발표 자료에 넣기 쉽게 모델 성능표와 혼동행렬을 정리한 표이다. 네 모델의 주요 지표와 TP, FP, FN, TN을 한 화면에서 비교할 수 있다.

벤치마크 용어 설명
- AUROC와 PR-AUC: 임계값과 무관하게 모델의 구분 성능을 보는 지표이다.
- Precision, recall, F1: 0.590 임계값에서의 High 예측 성능이다.
- Confusion matrix: 예측 결과를 TP, FP, FN, TN으로 나눈 표이다.

결과 해석
보고서 표 기준으로 Logistic Regression은 recall이 가장 높고, XGBoost는 PR-AUC가 가장 높다. LightGBM은 두 지표가 모두 중간 이상이며, Random Forest는 보수적으로 예측해 recall이 낮다.

보고서에 넣을 문장
"보고서용 표에서는 전문가 상담 후 변수 구성에서 Logistic Regression, XGBoost, LightGBM이 서로 비슷한 후보군이며, 목적에 따라 recall 또는 PR-AUC를 중심으로 해석할 수 있음을 제시하였다."

주의점
표에 들어간 결과는 모두 0.590 임계값 기준이다. 기존 ml 폴더의 모델별 자동 선택 임계값 결과와 직접 비교할 때는 임계값 차이를 설명해야 한다.
