# 보정 곡선 - Calibration Curves

이미지 파일: calibration_curves.png

공통 벤치마크 기준
- 데이터: 전문가 상담 후 재전처리한 지역사회건강조사 데이터, 19세 이상 39세 이하 청년층 표본 134,672건.
- 참고 라벨: PHQ-9 합산점수 10점 이상을 Reference high, 10점 미만을 Reference low로 정의하였다.
- 클래스 분포: Reference high는 5,468건으로 전체의 4.1%이고, Reference low는 129,204건으로 전체의 95.9%이다.
- 변수 변경: 가구원 수는 인원수 그대로 쓰지 않고 1인 가구 여부(single_person_household)로 재범주화하였다.
- 제외 변수: 주관적 건강인식, 체형인식(마름/비만), 아침식사 빈도, 교육수준은 전문가 상담 내용을 반영해 입력 변수에서 제외하였다.
- 벤치마크 목적: Logistic Regression, Random Forest, XGBoost, LightGBM을 같은 데이터 분할과 같은 전처리 조건에서 비교해 어떤 모델이 선별 목적에 적합한지 확인하는 것이다.
- 임계값 기준: 이번 전문가 상담 후 실험에서는 모든 모델의 예측 확률 임계값을 0.590으로 고정하였다.
- 해석 주의: 이 폴더의 결과는 기존 ml 폴더를 대체한 것이 아니라, 전문가 상담 이후 변수 구성을 바꿔 다시 돌린 별도 비교 실험이다.

그림이 보여주는 것
이 그림은 모델이 출력한 예측 확률이 실제 Reference high 비율과 얼마나 잘 맞는지 보여준다. 예를 들어 모델이 0.20 정도의 위험 확률을 준 집단에서 실제 Reference high 비율도 20%에 가까우면 보정이 잘 된 모델이라고 볼 수 있다.

벤치마크 용어 설명
- Calibration: 모델이 말한 확률과 실제 발생 비율이 얼마나 가까운지 보는 개념이다.
- Brier score: 예측 확률과 실제 정답 사이의 오차를 요약한 값이다. 낮을수록 좋다.
- 대각선 기준선: 예측 확률과 실제 비율이 완전히 일치하는 이상적인 선이다.

결과 해석
이번 데이터는 Reference high 비율이 4.1%로 낮기 때문에, 확률값 자체를 진단 확률처럼 해석하기보다는 모델 간 보정 상태를 비교하는 보조 자료로 보는 것이 적절하다.

보고서에 넣을 문장
"보정 곡선은 모델의 예측 확률이 실제 Reference high 비율과 얼마나 일치하는지 확인하기 위한 보조 지표로 사용하였다."

주의점
이 그림은 모델이 맞고 틀렸는지를 직접 보여주는 그래프가 아니다. 확률값의 신뢰도를 보는 자료이며, 최종 모델 판단은 AUROC, PR-AUC, recall, F1, 혼동행렬과 함께 해석해야 한다.
