# LightGBM 결과 그룹별 기여도

이미지 파일: lightgbm_outcome_group_contributions.png

공통 벤치마크 기준
- 데이터: 전문가 상담 후 재전처리한 지역사회건강조사 데이터, 19세 이상 39세 이하 청년층 표본 134,672건.
- 참고 라벨: PHQ-9 합산점수 10점 이상을 Reference high, 10점 미만을 Reference low로 정의하였다.
- 클래스 분포: Reference high는 5,468건으로 전체의 4.1%이고, Reference low는 129,204건으로 전체의 95.9%이다.
- 변수 변경: 가구원 수는 인원수 그대로 쓰지 않고 1인 가구 여부(single_person_household)로 재범주화하였다.
- 제외 변수: 주관적 건강인식, 체형인식(마름/비만), 아침식사 빈도, 교육수준은 전문가 상담 내용을 반영해 입력 변수에서 제외하였다.
- 벤치마크 목적: Logistic Regression, Random Forest, XGBoost, LightGBM을 같은 데이터 분할과 같은 전처리 조건에서 비교해 어떤 모델이 선별 목적에 적합한지 확인하는 것이다.
- 임계값 기준: 이번 전문가 상담 후 실험에서는 모든 모델의 예측 확률 임계값을 0.590으로 고정하였다.
- 해석 주의: 이 폴더의 결과는 기존 ml 폴더를 대체한 것이 아니라, 전문가 상담 이후 변수 구성을 바꿔 다시 돌린 별도 비교 실험이다.

그림이 보여주는 것
이 그림은 LightGBM 예측 결과를 TP, FP, FN, TN 네 그룹으로 나눈 뒤, 각 그룹에서 변수들이 예측값을 어느 방향으로 밀었는지 평균 SHAP 값으로 보여준다.

벤치마크 용어 설명
- SHAP: 각 변수가 개별 예측값을 얼마나 올리거나 낮췄는지 설명하는 방법이다.
- TP/FP/FN/TN: 맞춘 High, 잘못 올린 High, 놓친 High, 맞춘 Low를 뜻한다.
- Signed contribution: 양수면 High 예측 쪽으로, 음수면 Low 예측 쪽으로 밀었다는 의미이다.

결과 해석
이 그래프는 LightGBM이 어떤 변수 조합에서 High를 잘 잡거나 놓치는지 살펴보기 위한 설명용 자료이다. 특히 stress_level, sex, smoking_status 같은 변수가 결과 그룹별로 어떻게 다르게 작동하는지 확인할 수 있다.

보고서에 넣을 문장
"LightGBM 결과 그룹별 SHAP 분석을 통해 맞춘 사례와 놓친 사례에서 변수 기여 패턴이 어떻게 다른지 확인하였다."

주의점
그룹 평균 SHAP이므로 개인별 설명과는 다르다. 개별 사용자의 예측 이유를 설명하려면 해당 개인 행의 SHAP 값을 별도로 보아야 한다.
